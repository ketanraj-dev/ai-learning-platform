# AI-Powered Multi-Modal Learning Platform

A full-stack academic project вЂ” an intelligent web-based learning platform with face recognition login, voice interaction, adaptive quizzes, and AI-powered tutoring for Data Science and AI/ML topics.

---

## Features

- **Multi-Modal Authentication** вЂ” Email/password login + Face recognition login via webcam
- **AI Doubt Solver** вЂ” Real-time streaming chat powered by GPT-4o-mini, scoped to DS/AI-ML
- **Voice Interaction** вЂ” Speech-to-text via OpenAI Whisper + Text-to-speech via browser API
- **5 DS/AI-ML Courses** вЂ” Python, Machine Learning, Deep Learning, NLP, Statistics
- **Adaptive Quizzes** вЂ” Easy/Medium/Hard difficulty auto-adjusts based on your score
- **Mock Tests** вЂ” Full-length mixed-difficulty assessments
- **Analytics Dashboard** вЂ” Topic accuracy, weekly trends, strength/weakness heatmaps
- **AI Recommendations** вЂ” Personalised study plans generated by GPT

## Setup & Installation

### Prerequisites

- Python 3.11 or 3.12
- Node.js 18+
- Visual C++ Build Tools (Windows) вЂ” required for dlib/face_recognition
- CMake (Windows)
- OpenAI API key

---

### Backend Setup

```bash
# 1. Create and activate virtual environment
cd backend
python -m venv venv

# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

# 2. Install face_recognition dependencies first (Windows)
pip install cmake
pip install dlib    # Use prebuilt wheel for Python 3.12:
                    # https://github.com/z-mahmud22/Dlib_Windows_Python3.x/releases

pip install face_recognition

# 3. Fix face_recognition_models for Python 3.12 (Windows)
# Open this file in notepad:
# venv\Lib\site-packages\face_recognition_models\__init__.py
# Replace: from pkg_resources import resource_filename
# With:    import os; resource_filename = lambda pkg, path: os.path.join(os.path.dirname(__file__), path)

# 4. Install all dependencies
pip install -r requirements.txt

# 5. Create .env file
cp .env.example .env
# Edit .env and add your OpenAI API key

# 6. Start the backend
uvicorn app.main:app --reload
```

### Frontend Setup

```bash
cd frontend

# Install dependencies
npm install
npm install recharts zustand react-router-dom @tanstack/react-query axios lucide-react

# Install Tailwind v3
npm install -D tailwindcss@3.4.4 postcss autoprefixer
npx tailwindcss init -p

# Start the frontend
npm run dev
```

### Seed the Database

```bash
# In a new terminal (with venv active)
cd backend
python app/db/seed.py
```

This creates 5 DS/AI-ML courses, 20 lessons, and 56 questions.

## Known Issues & Notes

- Face recognition requires good lighting and a front-facing camera
- Whisper STT requires a valid OpenAI API key
- Face recognition is slower on first call (dlib loads model files)
- SQLite is used for development вЂ” switch to PostgreSQL for production by changing `DATABASE_URL` in `.env`
